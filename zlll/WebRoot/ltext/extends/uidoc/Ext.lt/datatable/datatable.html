datatable

命名空间：Ext.lt.datatable35

说明：

datatable组件是为了大数据量浏览器展现而设计的基本表格组件。主要解决浏览器采用Table标签展现大数据量表格时表现的性能问题，并实现了客户端多层表头、排序、表格搜索、行列锁定等常见表格需求。
datatable组件使用Ext.lt.recordset作为客户端数据存储结构。可以有效减少数据量的传输，提高网络利用率。
datatable采用可视区域渲染技术，较3.0版本采用的预渲染技术渲染速度受数据量影响更小，生成速度更快，并且避免在浏览器中渲染过多内容，造成浏览器占用内存过大性能下降严重的问题，系统缺省实现次数据渲染时间在30ms以内，以保证用户再拖拽操作时操作流畅。

使用时需要在页面中引用以下文件
<link rel="stylesheet" type="text/css" href="/ltext/datatabletheme.css" />
<link rel="stylesheet" type="text/css" href="/ltext/datatabletheme35.css" />
<script src="/ltext/ltext_core.js"></script>
<script src="/ltext/datatable3.0.js"></script>

代码如下所示：
var rs=new Ext.lt.recordset(column:[],datas:[]);
var table=new Ext.lt.datatable35(rs);
table.draw(HTMLElement);

数据集对象Ext.lt.recordset对象实例，必填属性

组件版本号，3.0或3.5

type

组件类型，datatable

方法说明

setCols

设置表格列对象，数组形式。如果不设置，组件将通过数据集自动创建列设置。datatable中已经默认实现编码、单选、多选列，可以通过columns属性直接使用

table.setCols([
	table.columns.seq,
	table.columns.checkbox,
	{name:'code',alias:'用户编码',type:'S'}
]);
参数名 数据类型 说明
id int 列索引，不设置时系统会通过对象内置序列生成
name String 列名，对应数据集的属性名，区分大小写。系统默认值为 "col"+id
alias String 列别名或显示名称。如果不设置，系统默认使用name属性的值作为别名
style String 列样式单内容，通过该属性控制列显示样式
display boolean 是否显示该列
minwidth int 列最小宽度，用户调整列宽时能设置的最小列宽。组件默认最小是50像素最大300像素
width int 列宽度设置，不得小于minwidth，如果小于minwidth组件会以minwidth为准。width属性值集范围定义如下：
大于1 表示实际宽度值
0-1之间 按表格整体宽度的百分比自动匹配宽度
小于0 按表格整体宽度的相对值设置宽度。如表格整体宽300，列宽设置为-230，列实际宽度为70
'auto' 按表格整体宽度自动适应宽度
以上设置在用户手工调整宽度后失效，以用户调整宽度为准
datatype String 数据类型，目前组件支持的数据类型包括：S-字符串 N-数字 D-日期 P-百分比 F-浮点数型 I-整数型。默认为字符串
mapper Object mapper用于生成对照数据。比如平台业务数据中基础数据项全部使用itemid保存，将id转换name的工作可以通过formart属性进行转换。数据对照表通过mapper属性来设置。mapper的对象结构如下：
{default_column:缺省列默认第一列,columns:[n1,n2,……]列名,datas:[[],[]……]对照数据} 只有类型为S时生效
addMapperDatas [[]] 对于文本列对象会自动追加addMapperDatas。可以通过该接口追加翻译数据。数据结构是二维数组，数组内数据顺序必须与mapper对象中columns数组中顺序一致
getMapperData string/number/object 对于文本列对象会自动追加addMapperDatas。可以通过该方法查询指定值的mapping对象。可以直接传入该列属性的值，也可以传整行值对象进去
format String/object 输出格式，针对不同的数据类型设置有不同的格式设置方法：
数值型(N)：{unit:int 计量单位,dotname:int 小数位,qfw:boolean 是否显示}
文本型(S)：#n1-#n2
日期型(D)：'YYYY年MM月DD日' 输出格式，默认输出格式为'YYYY年MM月DD日' 输入日期格式必须为'YYYYMMDD'
输出格式请参见示例 百分比型(P)：值为0到1之间的浮点数，输出格式为两位整数一位小数，以百分号结尾 浮点数型(F)：可以控制千分位和小数点后位数，不受计量单位控制 整数型(I)：可以控制是否显示千分位，输出时强制取整。
fn function fn函数是生成表格单元格HTML代码的函数，默认实现如下所示：
function(i,j,rs,value){
	if(value==null || value=='null') value='';
	return ['<span title="',value,'" >',value,'</span>'].join('');
}
可以覆盖函数内部实现，按需生成所需要的格式和内容。
onbeforesort function(colconfig,table,asc) 排序事件，当点击表头表格开始排序之前时触发，方法返回false时将取消排序动作。
filter boolean 默认false，当设置为true时，表头列名前追加漏斗标志，点击该标志可以弹出列筛选窗口。对列表显示内容进行过滤
head Array[String] 需要生成错层表头时首先需要调用datatable的drawMultiHead设置为true。 表头按列设置，通过表头名称的数组来设置该列上层表头的名称，如果需要纵向合并，可以设置成相同的名字。相邻两列的表头名一致的话，组件自动进行横向合并。例如，直接支付和授权支付上增加一个集中支付的表头：col1={name:'直接支付',head:['集中支付']} col2={name,'授权支付',head:['集中支付']}。详见实例
_dt datable 组件生成后，自动追加到列对象上的表格对象引用

getCols

获取DataTable列对象集合，返回数组
datatble.getCols();

getCol(int/string)

获取DataTable列对象，可以通过列序号或者列名查询
datatable.getCol(1); datatable.getCol(name);

draw

将表格渲染到指定HTML对象中，一般为DIV
datatable.draw(HTMLElement)

position

按照表格生成的HTML元素的大小，重新计算表格内个区域的位置
datatable.position()

resize

按照指定的宽度和高度重新绘制表格
datatable.resize(width,height)

clockColumnSize

设置锁定列数，参数值大于1
datatable.clockColumnSize(2)

clockRowSize

设置锁定行数，参数值大于1
datatable.clockRowSize(2)

setClockRowSet

设置锁定行数据集，数据集可以是多行，类型必须是Ext.lt.recordset。使用这种方法可以通过独立的锁定行数据生成表格的锁定行。简化使用单一结果集生成锁定行时处理排序的过程。但是由于合计行的数据与表格数据集是两个数据集，因此，对表格数据的操作，如选中、编辑等，都不起作用。
table.setClockRowSet(new Ext.lt.recordset({columns:['userid','code','name','organtype_name'],datas:[['test1','testcode1','testname1','testorgname1'],['test2','testcode2','testname2','testorgname2']]}));

getClockRowSet

获取表格锁定行数据，如果表格没有设置锁定行数据集则返回null； table.getClockRowSet();

reflash

重新渲染数据区
datatable.reflash()

redraw

重绘整个表格
datatable.redraw()

mergeColumn

合并列设置,参数为整数或整数数组，数组的值表格表格显示是从左到右合并的列数。例如[2,1,3]表示1、2列合并显示，4、5、6列合并显示，其余列单独显示
datatable.mergeColumn([2,1,3])

getZoom

获取表格内的放缩比例，默认是1
datatable.getZoom()

setZoom

设置表格内文字显示的比例，参数值大于0
datatable.setZoom(1.2)

headsort

是否允许排序功能，默认为不支持，系统内置的序列、单选、多选列不支持排序。datatable支持三态排序功能，即第一次点击升序排列，第二次点击降序排列，第三次点击回复数据默认顺序
datatable.headsort(true)

clearCols

清除表格列设置
datatable.clearCols()

addMapperDatas

向指定列对象中追加对照数据，数据格式为二维数组，不需要指定列名，但数据顺序必须与列对象中mapper中设置的一直
datatable.addMapperDatas(columnname,datas)

setClassName

设置样式单名称，缺省样式单名为dt_tagdiv。目前组件支持的样式有dt_tagdiv、dttheme_ifmis、dttheme_beijing、dttheme_inner、dttheme_spring（春天主题）
datatable.setClassName('dttheme_ifmis');

setSelectClassName

设置选中行颜色，需要配置单选列或多选列
datatable.setSelectClassName('red');

clickselected

设置行是否可以点击变色，背景色通过setSelectClassName方法设置。方法参数true表示单行选中，'n'表示支持多行选中。
datatable.clickselected(true);
datatable.clickselected('n');

setAllowClock

允许在列上进行列锁定操作，默认为true。可锁定列会在列名后面追加一个小锁的标记
datatable.setAllowClock(false);

setAllowHeadWidth

是否允许调整列宽，默认为true，设置为true后，鼠标滑动到表头单元格右侧边线是会出现调整列宽的图标
datatable.setAllHeadWidth(false);

setAllowLineHeight

是否允许调整行高，默认为 false，设置为true后，鼠标滑动到单元格行线附近会出现调整行高的图标
datatable.setAllowLineHeight(false);

drawMultiHead

是否绘制多层表头，默认为false。设置为true后，将参考列对象上的head属性绘制表头
datatable.drawMultiHead(true);

setLayout

集成布局管理
datatable.setLayout();

setMouselight

设置表格鼠标高亮颜色，默认空白色。如'#eee' 'red'
datatable.setMouselight('#eee');

mousedrag

是否支持鼠标拖拽。默认为ture 设置为false时表格不支持鼠标拖拽。datatable的特色功能，通过鼠标拖拽的方式移动数据区，简化表格横向拖动操作。datatable3.5组件支持shift+鼠标滚轮的方式进行表格横向滚动。
datatable.mousedrag(false);

setMaxData

设置表格显示最大行数，一般用于分页控制
datatable.setMaxData(10000);

gatMaxData

获取表格显示最大行数
datatable.gatMaxData();

setFilter

设置过滤条件，设置格式为{key:string/number/array,…}，设置后DataTable只显示符合条件的数据
datatable.setFilter({code:['101']});

clearFilter

清除数据过滤条件
datatable.clearFilter();

remove

删除表格中选中的数据,参数为删除条件，参数为null时删除所有数据。参数对象格式为{key:string/number/array,…}
datatable.remove({expfunc:['201','202']});

setKeysearch

是否开启键盘快速定位功能，默认false。功能启用后可以通过shift+sh激活搜索窗口，也可以通过setSearchWord接口自己实现
datatable.setKeysearch(true);

setSearchWord

设置搜索关键词，搜索关键词。关键词搜索采取全文检索方式，匹配时不区分字段匹配所有可见文本。
datatable.setSearchWord('预算');

clearSearchWord

清除搜索关键词
datatable.clearSearchWord();

goPreSearchWord

从当前选中搜索词向前搜索并定位搜索词。如果已经到达表格最上方，则弹出“已经搜索完毕”的提示框，再次调用将从表格末行重新搜索
datatable.goPreSearchWord();

goNextSearchWord

从当前选中搜索词向后搜索并定位搜索词。如果已经到达表格最下方，则弹出“已经搜索完毕”的提示框，再次调用将从表格首行重新搜索
datatable.goNextSearchWord();

goto

直接跳转到指定行数
datatable.goto(100);

gotoCell

跳转到指定单元格，跳转结束后回调指定方法，并将单元格对象作为参数反馈
datatable.gotoCell(100,10,function(){cellobj});

setAlign

表格内部对齐方式，当表格的列比较少时，可以通过方法设置表格数据区显示的位置。默认为居左。
datatable.setAlign('center');

setRecordset

设置表格数据集
datatable.setRecordset(new Ext.lt.recordset);

showLine

设置纵向滚动条拖动提示框显示内容。方法参数为 boolean/String/function。改功能默认关闭。设置为true后显示当前显示第一行行号。也可以指定列名，显示对应列的内容，或通过函数自定义显示内容。传入函数为function(d//对应界面第一行数据集){return ''// 显示内容}
datatable.showLine(function(d){return d['name']});

setMouselightCell

设置是否启用鼠标选中单元格效果。选中单元格样式通过.dt_mousecell控制。
datatable.setMouselightCell(true);

setColumndrag

设置是否可以通过拖动表头来调整列顺序，默认false。设置为true后可以拖动表头调整列顺序。鼠标松开后表格对象会发送Ext.lt.message.send("datatable","swapcolumn",[table])消息。消息执行后会刷新表格。
datatable.allowColumndrag(true);

swapColumn

将列设置到指定的位置，例如：把表格第3列放到第5列的位置。在调用的时候表格会发出Ext.lt.message.send("datatable","swapcolumn",[table])消息。该消息在列对象调整完成后发出，消息执行结束后会重绘表格。
datatable.swapColumn(2,4);

setBreakWord

设置表格是否可以折行显示，boolean值，默认false及不能折行。设置为true后表格内的内容可以折行显示
datatable.setBreakWord(true);

setDisplayColumn

设置显示列，可以通过列名或列对象设置。
datatable.setDisplayColumn('name');
datatable.setDisplayColumn(['name',{name:'expfunc'}]);

setHiddenColumn

设置隐藏列，可以通过列名或列对象设置。
datatable.setHiddenColumn('name');
datatable.setHiddenColumn(['name',{name:'expfunc'}]);

sortColumn

表格排序方法，参数可以是过列对象、列序号、列名
datatable.sortColumn('name');
datatable.sortColumn(1);
datatable.sortColumn(datatable.getCols()[3]);

sortData(col/name/colid,asc)

表格数据排序，参数可以是过列对象、列序号、列名，第二个参数控制排序顺序true为升序，false为降序，null为数据默认顺序。
datatable.sortData('name',true);
datatable.sortData(1,false);
datatable.sortData(datatable.getCols()[3],null);

saveColumnWidth(boolean)

设置是否将列宽保存到客户端，下次创建表格时仍然使用用户最后一次调整的列宽。该功能只有发布到域中才能实现。
datatable.setUserWid(true);

事件说明

表格默认支持的事件类型包括：onclick,onselect,ondblclick,onmousemove,onmouseout,onmouseenter,onmousein,onmouseover,onmouseup,onmousedown,onselectstart,onmousewheel,onkeypress,onkeyup,onscroll,onmousewheel。
表格事件分为三类：单元格事件、列事件、表头事件。单元格事件可以通过onEvent(eventname,function)方法处理各种事件。列事件在列对象上通过事件名直接定义。表头事件在列对象上通过onhead+事件名定义
回调函数参数说明
表头事件：fu(table,el,c,d) table-datatable对象;el-产生时间的单元格对象;c-列号;单元格所在行对应的数据对象
列事件：fn(table,el,l,c,d) table-datatable对象;el-产生时间的单元格对象;l-行号;c-列号;单元格所在行对应的数据对象
单元格事件：fn(table,el,l,c,d) table-datatable对象;el-产生时间的单元格对象;l-行号;c-列号;单元格所在行对应的数据对象

消息说明

当用户与datatable组件进行交互时，除了会产生各种处理事件之外，还会产生各种消息。
可以使用Ext.lt.message.hook('datatable','消息类型',处理函数)接收datatab消息
来源对象名：datatable

消息类型,参数对象结构
onclick,[datatable对象,消息来源对象,行号,列号,点击的数据集对象]
